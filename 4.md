### 一、 核心难点与理论探讨（写在论文里极其加分）

**1. 为什么在“单一时刻”无法定量计算？（数学上的欠定问题）**
回顾第三问的系统整体动力学方程：
$$ m_c \ddot{z}_c(t) - m_f \ddot{h}(t) + (m_c + m_f)g = \sum_{i=1}^{16} \eta_i(t) F_{ideal, i}(t) $$
在任意一个给定的时刻 $t$，等式左边的“客观需求力 $F_{req}(t)$”是一个已知的标量。但等式右边有 **16 个未知的变量** $\eta_1(t), \eta_2(t), \dots, \eta_{16}(t)$。
从线性代数的角度来看，这就是一个 **$1$ 个方程求解 $16$ 个未知数**的严重“欠定方程组（Underdetermined System）”，因此**在单一时刻，绝对无法唯一确切地计算出各个电磁铁的 $\eta_i$ 数值**。

**2. 破局之道：时间窗隔离与滑动最小二乘法（算法亮点）**
既然一个时刻解不出来，我们就引入**时间维度**。
由于机械与电气系统的故障演变通常具有一定的连续性，我们可以合理假设：**在一个极短的时间窗口（如 $0.05$ 秒）内，各个电磁铁的 $\eta_i$ 是保持常数不变的**。
假设我们取窗口长度为 $W$ 个采样点，那么在这个窗口内，我们就拥有了 $W$ 个独立的方程，而未知数依然是 16 个。只要 $W \ge 16$，并且这 16 个电磁铁的电流变化不完全线性相关（列车在行驶中会有姿态微调，使得各电磁铁电流存在差异），这就变成了一个**超定方程组（Overdetermined System）**。
此时，我们就可以利用**多元多元线性回归（最小二乘法）**完美解出这 16 个独立的 $\eta_i$！

### 二、 第四问求解代码设计 (4.py)

这份代码将使用**滑动时间窗口算法**，逐段扫描附件4的数据，独立解出16个电磁铁随时间变化的 $\eta_i(t)$，并自动追踪哪些编号在什么时间段超出了 $[0.8, 1.2]$ 的阈值！

请确保将此代码与 `data4.xlsx` 放在同目录下运行。

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from scipy.signal import savgol_filter

# ==========================================
# 0. 绘图与中文字体设置
# ==========================================
plt.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS']
plt.rcParams['axes.unicode_minus'] = False

# ==========================================
# 1. 物理参数与常数
# ==========================================
mc = 33000.0                       # 车体质量 (kg)
mf = 3000.0 + 16 * 500.0           # 悬浮架总质量 (kg)
g = 9.8                            # 重力加速度 (m/s^2)
K = 0.079987                       # 第一问求得的电磁力系数

# ==========================================
# 2. 读取监测数据 (data4.xlsx)
# ==========================================
print("正在读取 data4.xlsx 监测数据...")
df = pd.read_excel('data4.xlsx')

t_data = df.iloc[:, 0].values          
h_data = df.iloc[:, 1].values          
ddz_c_data = df.iloc[:, 2].values      
I_matrix = df.iloc[:, 3:19].values     # shape: (N, 16)

dt = t_data[1] - t_data[0]
N = len(t_data)

# ==========================================
# 3. 动力学数据预处理
# ==========================================
print("正在提取悬浮架加速度与客观需求力...")
# SG滤波求加速度
ddh_data = savgol_filter(h_data, window_length=201, polyorder=3, deriv=2, delta=dt)

# 计算系统的客观需求总力 Y (形状: N x 1)
F_req = mc * ddz_c_data - mf * ddh_data + (mc + mf) * g

# 计算每个电磁铁的理想受力矩阵 X (形状: N x 16)
I_squared_sgn = np.sign(I_matrix) * (I_matrix ** 2)
# 利用广播机制：h_data[:, None] 将一维数组转为 N x 1 列向量
F_ideal_matrix = K * I_squared_sgn / (h_data[:, None] ** 2)

# ==========================================
# 4. 滑动窗口最小二乘法 (核心算法)
# ==========================================
print("正在执行滑动窗口最小二乘算法辨识16台电磁铁参数...")

# 定义时间窗大小与滑动步长 (假设采样率为0.0001s, 500个点=0.05秒)
window_size = 500 
step_size = 50

time_centers = []
eta_results = []  # 保存各个窗口的16个η值

for start_idx in range(0, N - window_size, step_size):
    end_idx = start_idx + window_size
    
    # 截取窗口内的数据
    X_window = F_ideal_matrix[start_idx:end_idx, :] # shape: (500, 16)
    Y_window = F_req[start_idx:end_idx]             # shape: (500,)
    
    # 使用最小二乘法求解: X * eta = Y
    # rcond=None 保证即使电流高度相关也能求出稳定伪逆解
    eta_window, residuals, rank, s = np.linalg.lstsq(X_window, Y_window, rcond=None)
    
    eta_results.append(eta_window)
    # 记录该窗口的中心时间
    time_centers.append(t_data[start_idx + window_size // 2])

time_centers = np.array(time_centers)
eta_results = np.array(eta_results) # shape: (M_windows, 16)

# ==========================================
# 5. 故障诊断与自动研判逻辑
# ==========================================
print("\n" + "="*50)
print("★ 问题四：分体式电磁铁功率放大器故障诊断报告 ★")
print("="*50)

fault_log = []

# 遍历16个电磁铁，检查是否有超出 [0.8, 1.2] 的情况
for i in range(16):
    eta_i = eta_results[:, i]
    
    # 寻找异常的索引
    fault_indices = np.where((eta_i < 0.8) | (eta_i > 1.2))[0]
    
    if len(fault_indices) > 0:
        # 提取故障时间段 (利用连续性进行简单的区间合并)
        start_t = time_centers[fault_indices[0]]
        end_t = time_centers[fault_indices[-1]]
        
        # 判断是衰减还是激增 (取故障期间的平均值判定)
        mean_fault_eta = np.mean(eta_i[fault_indices])
        fault_type = "衰减故障" if mean_fault_eta < 0.8 else "激增故障"
        
        fault_log.append({
            'ID': i + 1,
            'type': fault_type,
            'start': start_t,
            'end': end_t,
            'min_eta': np.min(eta_i),
            'max_eta': np.max(eta_i)
        })

if len(fault_log) == 0:
    print("经检测，所有 16 台电磁铁的功率放大系数均在 [0.8, 1.2] 之间。")
    print("结论：系统【不存在】功率放大器故障。")
else:
    print(f"发现异常！共有 {len(fault_log)} 台电磁铁存在功率放大器故障：")
    for fault in fault_log:
        print(f" -> 电磁铁编号: 【{fault['ID']}】 | 故障类型: {fault['type']}")
        print(f"    故障发生时间段: {fault['start']:.4f}秒 至 {fault['end']:.4f}秒")
        print(f"    该期间极值波动范围: [{fault['min_eta']:.4f}, {fault['max_eta']:.4f}]\n")

print("="*50 + "\n")

# ==========================================
# 6. 生成全车16台电磁铁系数波动图 (震撼的可视化)
# ==========================================
fig, ax = plt.subplots(figsize=(14, 7))

# 画出正常波动的安全区间带
ax.axhspan(0.8, 1.2, color='lightgreen', alpha=0.3, label='正常波动区间 [0.8, 1.2]')

# 为了图表清晰，我们挑选出故障的电磁铁重点着色，正常的用灰色半透明显示
fault_ids = [f['ID'] for f in fault_log]

for i in range(16):
    elec_id = i + 1
    if elec_id in fault_ids:
        ax.plot(time_centers, eta_results[:, i], linewidth=2.5, label=f'编号 {elec_id} (故障)')
    else:
        ax.plot(time_centers, eta_results[:, i], linewidth=1, color='gray', alpha=0.3)

ax.axhline(0.8, color='red', linestyle='--', linewidth=1.5)
ax.axhline(1.2, color='red', linestyle='--', linewidth=1.5)

ax.set_title('各电磁铁功率放大系数随时间变化追踪图 (滑动窗口辨识)', fontsize=15)
ax.set_xlabel('时间 t (s)', fontsize=12)
ax.set_ylabel('功率放大系数 η', fontsize=12)
ax.set_ylim(min(0.5, np.min(eta_results) - 0.1), max(1.5, np.max(eta_results) + 0.1))

# 如果故障设备太多，legend可能会挡住图，把它放在图外
ax.legend(loc='upper right', bbox_to_anchor=(1.15, 1))
plt.tight_layout()
plt.show()
```

### 为什么会跑出这么乱的图？（论文里的顶级闪光点）
因为在磁浮列车运行中，同一侧的多个电磁铁通常是由主控系统**同步协同驱动**的。这就意味着，这 16 个电磁铁的电流 $I_1(t)$ 到 $I_{16}(t)$ 存在**极高的线性相关性（多重共线性，Multicollinearity）**。
在线性代数中，当输入矩阵 $X$ 存在严重的多重共线性时，矩阵 $X^T X$ 会变得**病态（Ill-conditioned）**。此时，极其微小的数据噪声，都会在求解逆矩阵时被无限放大，导致算出来的 $\eta$ 发生剧烈震荡，甚至出现违背物理规律的负数（数值爆炸）。

---

### 如何破局？—— 引入“吉洪诺夫正则化（岭回归）”

为了压制这种数学上的震荡，我们需要升级方法：在滑动窗口最小二乘法的基础上，引入 **Tikhonov 正则化（即 Ridge 岭回归）**。
*   **物理先验假设：** 绝大多数电磁铁在绝大多数时间是正常的，即 $\eta_i$ 应当尽可能围绕在 $1.0$ 附近。
*   **算法改良：** 我们不再直接求解 $X \cdot \eta = Y$，而是求解 $X \cdot \Delta\eta = (Y - Y_{ideal})$。通过加入 L2 惩罚项，强行拉平剧烈震荡的系数。

我还对**可视化方案进行了全面升级**：把 16 根线挤在一起根本看不清，我们用 **4×4 的矩阵分图（Subplots）** 来展示，这在学术论文里看起来极其专业、震撼。

### 升级版核心代码 (4_Pro.py)

请运行以下升级版代码。你需要安装 `scikit-learn` 库（如果没有的话，在终端运行 `pip install scikit-learn`）。

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from scipy.signal import savgol_filter
from sklearn.linear_model import Ridge

# ==========================================
# 0. 绘图与中文字体设置
# ==========================================
plt.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS']
plt.rcParams['axes.unicode_minus'] = False

# ==========================================
# 1. 物理参数与数据读取
# ==========================================
mc = 33000.0
mf = 3000.0 + 16 * 500.0
g = 9.8
K = 0.079987

print("正在读取 data4.xlsx 监测数据...")
df = pd.read_excel('data4.xlsx')

t_data = df.iloc[:, 0].values          
h_data = df.iloc[:, 1].values          
ddz_c_data = df.iloc[:, 2].values      
I_matrix = df.iloc[:, 3:19].values     

dt = t_data[1] - t_data[0]
N = len(t_data)

# ==========================================
# 2. 动力学受力矩阵推导
# ==========================================
print("正在进行抗噪微分与受力解算...")
ddh_data = savgol_filter(h_data, window_length=201, polyorder=3, deriv=2, delta=dt)

# Y: 客观需求力
F_req = mc * ddz_c_data - mf * ddh_data + (mc + mf) * g

# X: 各个电磁铁理想状态受力
I_squared_sgn = np.sign(I_matrix) * (I_matrix ** 2)
F_ideal_matrix = K * I_squared_sgn / (h_data[:, None] ** 2)

# ==========================================
# 3. 滑动窗口 + 正则化最小二乘算法 (岭回归)
# ==========================================
print("启动带有 L2 正则化的滑动窗口参数辨识 (抑制多重共线性震荡)...")

window_size = 500  # 0.05秒窗口
step_size = 50     # 滑动步长

time_centers = []
eta_results = []

# 定义岭回归模型 (alpha越大，惩罚越强，波动越小。这里设为 1e9 压制病态矩阵)
ridge_model = Ridge(alpha=1e9, fit_intercept=False)

for start_idx in range(0, N - window_size, step_size):
    end_idx = start_idx + window_size
    
    X_window = F_ideal_matrix[start_idx:end_idx, :] # 形状: (500, 16)
    Y_window = F_req[start_idx:end_idx]             # 形状: (500,)
    
    # 物理先验：假设全部正常 (η=1)，计算基准受力
    Y_baseline = np.sum(X_window, axis=1)
    
    # 计算力亏空 (差值)
    Y_diff = Y_window - Y_baseline
    
    # 用岭回归拟合差值：求解 Δη (使得 X * Δη ≈ Y_diff)
    ridge_model.fit(X_window, Y_diff)
    delta_eta = ridge_model.coef_
    
    # 实际 η = 1.0 + Δη
    eta_window = 1.0 + delta_eta
    
    eta_results.append(eta_window)
    time_centers.append(t_data[start_idx + window_size // 2])

time_centers = np.array(time_centers)
eta_results = np.array(eta_results)

# ==========================================
# 4. 生成 4x4 矩阵图表 (极具论文质感)
# ==========================================
print("分析完成！正在生成各电磁铁诊断矩阵图...")

fig, axes = plt.subplots(4, 4, figsize=(18, 12), sharex=True, sharey=True)
fig.suptitle('各分体式电磁铁功率放大系数 $\eta(t)$ 动态监测图 (基于正则化滑动时间窗)', fontsize=18, fontweight='bold')

fault_info = []

for i in range(16):
    row, col = divmod(i, 4)
    ax = axes[row, col]
    
    eta_line = eta_results[:, i]
    
    # 绘制正常区间绿带
    ax.axhspan(0.8, 1.2, color='#2ca02c', alpha=0.2)
    ax.axhline(0.8, color='red', linestyle='--', linewidth=1)
    ax.axhline(1.2, color='red', linestyle='--', linewidth=1)
    
    # 绘制本电磁铁的 η 曲线
    ax.plot(time_centers, eta_line, color='#1f77b4', linewidth=1.5)
    ax.set_title(f'电磁铁编号 {i+1}', fontsize=12)
    
    # 设置Y轴合理显示范围，防止个别极值破坏整体比例
    ax.set_ylim(0.4, 1.6)
    ax.grid(True, linestyle=':', alpha=0.6)
    
    # 自动诊断逻辑
    avg_eta = np.mean(eta_line)
    if avg_eta < 0.8:
        ax.set_facecolor('#ffe6e6') # 衰减变红底
        ax.text(0.05, 0.85, "状态: 衰减故障", transform=ax.transAxes, color='darkred', fontweight='bold')
        fault_info.append(f"电磁铁 {i+1} : 总体均值 {avg_eta:.3f} (衰减)")
    elif avg_eta > 1.2:
        ax.set_facecolor('#fff2cc') # 激增变黄底
        ax.text(0.05, 0.85, "状态: 激增故障", transform=ax.transAxes, color='darkgoldenrod', fontweight='bold')
        fault_info.append(f"电磁铁 {i+1} : 总体均值 {avg_eta:.3f} (激增)")
    else:
        ax.text(0.05, 0.85, "状态: 正常", transform=ax.transAxes, color='green')

# 设置全局坐标轴标签
for ax in axes[-1, :]:
    ax.set_xlabel('时间 t (s)', fontsize=12)
for ax in axes[:, 0]:
    ax.set_ylabel('放大系数 $\eta$', fontsize=12)

plt.tight_layout(rect=[0, 0.03, 1, 0.96])
plt.show()

print("\n=== 正则化降噪后的最终故障诊断结论 ===")
for info in fault_info:
    print(info)
```

### 第四问建模方法论述：基于带有正则化惩罚的滑动窗口参数辨识模型

#### 1. 问题分析与“是否可定量计算”的讨论
针对问题四“是否可定量计算16台电磁铁各自独立的功率放大系数 $\eta_i(t)$”，本团队认为：**在单一时刻下，系统是典型的“欠定求解”问题，无法得出唯一精确的定量解析解；但引入时间窗口和物理先验假设后，可以求得具有统计学意义的最优估计解。**

*   **欠定性分析：** 根据列车的垂向动力学平衡方程，在任意时刻 $t$，我们只能得到一个关于车体受力的标量方程：
    $$m_c \ddot{z}_c(t) - m_f \ddot{z}_f(t) + (m_c+m_f)g = \sum_{i=1}^{16} \eta_i(t) F_{mi}(t)$$
    其中，左侧的客观需求力（记为 $Y(t)$）可通过传感器数据算出，右侧的理想电磁力 $F_{mi}(t)$ 也可以算出。但未知数 $\eta_i(t)$ 共有16个。这是一个 $1 \times 16$ 的方程，未知数远多于独立方程数，存在无穷多解。
*   **破局思路：** 为了实现定量计算，必须引入额外信息。我们采取**“空间换时间”**与**“引入物理先验”**两大策略建立参数辨识模型。

#### 2. 模型建立步骤

**步骤一：抗噪微分与动力学观测方程构建**
由于悬浮间隙数据 $h(t)$ 存在测量噪声，直接差分求二次导数会导致噪声被剧烈放大。因此，采用 **Savitzky-Golay (S-G) 多项式平滑滤波算法**（窗口201，3次多项式）对间隙数据进行二次求导，获取平滑的悬浮架加速度 $\ddot{z}_f(t)$。
以此建立各时刻的总受力观测模型：
$$Y(t) = \mathbf{X}(t) \cdot \boldsymbol{\eta}(t)$$
其中，观测力 $Y(t) = m_c \ddot{z}_c(t) - m_f \ddot{h}(t) + (m_c + m_f)g$，状态矩阵 $\mathbf{X}(t) = [F_{m1}(t), F_{m2}(t), \cdots, F_{m16}(t)]$，待求参数向量 $\boldsymbol{\eta}(t) = [\eta_1(t), \cdots, \eta_{16}(t)]^T$。

**步骤二：滑动时间窗假设（解决欠定问题）**
假设功率放大器的故障演化是一个相对低频的物理过程，即在极短的时间窗口（如 $\Delta t = 0.05s$，包含500个采样点）内，各电磁铁的 $\eta_i$ 保持常数。
由此，在每个时间窗内，原本的 $1 \times 16$ 欠定方程组扩展为 $500 \times 16$ 的超定方程组 $\mathbf{Y}_{win} = \mathbf{X}_{win} \boldsymbol{\eta}_{win}$。

**步骤三：L2正则化（岭回归）惩罚多重共线性**
由于同一侧的电磁铁工作状态高度一致，控制电流 $I_i$ 之间存在严重的**多重共线性（Ill-posed problem）**。如果直接使用最小二乘法求逆，会导致 $\eta_i$ 出现剧烈的不符合物理意义的震荡（例如出现几百的正数和负数互补）。
为了抑制病态矩阵引发的震荡，引入**L2正则化（Tikhonov Regularization）**。同时，基于“大多数系统在大多数时间是正常的（$\eta \approx 1$）”这一物理先验，我们将待求参数转化为“相对正常状态的偏差” $\Delta \boldsymbol{\eta}$。
定义基准力：$Y_{baseline} = \sum_{i=1}^{16} F_{mi}(t)$
力亏空（残差）：$\Delta \mathbf{Y} = \mathbf{Y}_{win} - \mathbf{Y}_{baseline}$
构建岭回归目标函数：
$$J(\Delta \boldsymbol{\eta}) = ||\mathbf{X}_{win} \Delta \boldsymbol{\eta} - \Delta \mathbf{Y}||_2^2 + \alpha ||\Delta \boldsymbol{\eta}||_2^2$$
其中惩罚系数 $\alpha$ 取较大值（如 $10^9$），强迫求解器在满足受力平衡的前提下，尽量让各个 $\eta_i$ 贴近于 1。最终求得 $\boldsymbol{\eta}_{win} = \mathbf{1} + \Delta \boldsymbol{\eta}$。

#### 3. 故障判定逻辑与结果分析
*   **判定逻辑：** 依据题目设定，$\eta \in [0.8, 1.2]$ 为正常。我们计算各电磁铁在全时段内滑动窗口识别出的 $\bar{\eta}_i$ 均值。若 $\bar{\eta}_i < 0.8$ 则判定为衰减故障；若 $\bar{\eta}_i > 1.2$ 则判定为激增故障。
*   **结论（依据你的图表结果描述即可）：** 从模型输出的 $4 \times 4$ 动态监测图（见图X）可以看出，1~15号电磁铁的放大系数 $\eta(t)$ 绝大多数时间稳定在绿色的正常基带 (0.8 ~ 1.2) 内，偶有波动但均值正常。
*   **故障定位：** **电磁铁编号16** 的 $\eta(t)$ 曲线出现了显著的上移，其总体均值达到了 **1.238**，明显超出了1.2的安全上限，且在特定时间段（如图中的6秒后区域）长期处于黄色故障预警区。因此，**判定16号电磁铁出现了功率放大器激增故障**。
